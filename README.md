# 快速上手

## 相关技术

`react` , `ts`

## 安装

```bash
npm i lh-ui-react
```

or

```bash
yarn add lh-ui-react
```

## 使用示例

```bash
import { Hello } from 'lh-ui-react';

export default () => <Hello title="Hello React!" />;
```
